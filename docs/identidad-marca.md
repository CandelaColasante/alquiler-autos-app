## Paleta de colores
- Color: Azul oscuro, Código Hex: #424790, Uso en la app: header, footer, titulos, botones secundarios, paginación.
- Color: Azul claro, Código Hex: #dee4ff, Uso en la app: fondos secundarios (búsqueda), bordes de cards.
- Color: Blanco grisáseo, Código Hex: #fefeff, Uso en la app: fondo principal de toda la app.
- Color: Naranja, Código Hex: #eb5200, Uso en la app: botones principales, acciones importantes.
